# Golang parsing library for Vimoutliner files

Handles creating a Document Object Model (DOM) out of [Vimoutliner (VOTL)](https://github.com/vimoutliner/vimoutliner) 
files.

## License

Currently this library is using the BSDv3 license.

## Goals

To ba able to parse VOTL files and easily traverse them
for various tasks.
